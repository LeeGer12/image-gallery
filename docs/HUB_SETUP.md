# 中枢电脑配置指南

本指南教你把闲置电脑配置为图库服务器。完成后，公司所有同事可以通过局域网共享图库。

---

## 前提条件

- 一台 Windows 10 电脑（闲置的即可）
- 这台电脑和所有同事的电脑在同一个局域网（同一个路由器下）
- 建议用网线连接（比 Wi-Fi 稳定且快）

---

## 第一步：设置固定 IP

默认情况下路由器会给电脑分配动态 IP，重启后会变。必须设为固定 IP。

1. 按 `Win + I` 打开设置 → **网络和 Internet** → **以太网**
2. 点击你的网络连接 → **编辑** IP 设置
3. 改为 **手动**
4. 填写：
   - IP 地址：`192.168.1.100`（如果路由器是 192.168.0.x 段，就填 `192.168.0.100`）
   - 子网掩码：`255.255.255.0`
   - 网关：`192.168.1.1`（或 `192.168.0.1`，看你路由器背面写的）
   - 首选 DNS：`114.114.114.114`
   - 备用 DNS：`8.8.8.8`
5. 保存

> **怎么确认路由器网段？** 在电脑上按 `Win + R`，输入 `cmd` 回车，输入 `ipconfig` 回车。
> 找到"默认网关"那一行，如果显示 `192.168.1.1`，IP 就填 `192.168.1.100`。

---

## 第二步：安装 PostgreSQL

1. 打开浏览器，访问：https://www.postgresql.org/download/windows/
2. 点击 **Download the installer**，选择最新版本下载
3. 运行安装程序，全部保持默认，只有两处需要改：
   - **Password**：输入 `ImageGallery2026`（这是数据库超级管理员密码，请记住）
   - **Port**：保持默认 `5432`
4. 安装完成后，取消勾选 "Launch Stack Builder"，点 Finish

---

## 第三步：创建数据库和用户

1. 从开始菜单找到 **pgAdmin 4**，打开
2. 左侧树形菜单，右键 **PostgreSQL 16** → **Connect Server**
3. 输入密码 `ImageGallery2026`，点 Save
4. 左侧右键 **Databases** → **Create** → **Database**
   - Name 填：`imagegallery`
   - 点 Save
5. 左侧右键 **Login/Group Roles** → **Create** → **Login/Group Role**
   - General 页：Name 填 `gallery`
   - Definition 页：Password 填 `ImageGallery2026`
   - Privileges 页：勾选 Can login
   - 点 Save
6. 给 gallery 用户授权：
   - 左侧点开 **Databases** → 右键 **imagegallery** → **Properties**
   - 切到 **Privileges** 页
   - 点击右上角的 **+** 按钮
   - Role 选 `gallery`
   - 勾选 ALL（全部权限）
   - 点 Save

---

## 第四步：创建共享文件夹

1. 打开 **D 盘**（或空间最大的盘），新建文件夹命名为 `ImageGalleryStorage`
2. 右键这个文件夹 → **属性** → **共享** 选项卡
3. 点击 **共享...**
4. 在下拉框中输入 `Everyone`，点 **添加**
5. 把 Everyone 的权限改为 **读/写**
6. 点 **共享**，记录显示的网络路径（如 `\\DESKTOP-XXXX\ImageGalleryStorage`）
7. 点 **完成**

> **防火墙设置：** 如果同事连不上，可能是防火墙拦了。
> 控制面板 → Windows Defender 防火墙 → 允许应用通过 → 找到 **文件和打印机共享**，勾选专用和公用。

---

## 第五步：防火墙放行 PostgreSQL

1. 按 `Win + R`，输入 `wf.msc`，回车（打开高级安全防火墙）
2. 左侧点 **入站规则** → 右侧点 **新建规则**
3. 选 **端口** → 下一步
4. 选 **TCP**，特定端口填 `5432` → 下一步
5. 选 **允许连接** → 下一步
6. 勾选 **域**、**专用**、**公用** → 下一步
7. 名称填 `PostgreSQL`，点完成

---

## 验证

在同事的电脑上：

1. 按 `Win + R`，输入 `\\192.168.1.100\ImageGalleryStorage`，回车
2. 如果能打开文件夹，说明共享成功
3. 打开 ImageGallery 软件，输入 `192.168.1.100`，点连接
4. 显示"连接成功"即可使用

## 常见问题

**Q: 同事连不上怎么办？**
A: 按顺序检查：
1. 中枢电脑是否开机？
2. 两台电脑是否在同一网络？（互相 ping 一下）
3. 中枢电脑的 IP 是否变了？（重新检查固定 IP 设置）
4. 防火墙是否放行了 5432 端口和文件共享？

**Q: 中枢电脑重启后要做什么？**
A: 什么都不用做。PostgreSQL 和文件共享都是开机自启的。

**Q: 图片很多会变慢吗？**
A: 千兆局域网（1Gbps）下，浏览和下载图片速度很快。首次生成缩略图会慢一些，之后就快了。

**Q: 可以同时用中枢电脑做其他事吗？**
A: 可以。PostgreSQL 和文件共享占用资源很少，不影响正常使用。
