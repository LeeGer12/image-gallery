# Phase 5 多用户局域网共享 — 手动测试清单

测试日期_：_2026.05.29_

---

## 测试前准备

1. 确保 PostgreSQL 运行中，数据库已初始化
2. 确保 `users` 表为空（首次测试场景）：
   ```sql
   DELETE FROM operation_log;
   DELETE FROM users;
   ```
3. 如果之前测试过，重置 images 表的 imported_by 和 version：
   ```sql
   UPDATE images SET imported_by = NULL, version = 1;
   UPDATE albums SET created_by = NULL;
   ```
